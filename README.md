## 📊 Key Analysis Performed

### 1. Total Balance Per Customer
Calculated total account balances to understand customer value.

### 2. High-Value Transactions
Identified transactions above a defined threshold (e.g., > 10,000).

### 3. Failed Transactions
Analyzed unsuccessful transactions to detect system or fraud issues.

### 4. Fraud Detection
Examined flagged transactions and identified suspicious patterns.

### 5. Transaction Trends
Analyzed frequency and volume of transactions across accounts.
